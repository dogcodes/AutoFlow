## 🎯 使用说明

### 开发流程
1. **阅读项目概述** - 了解整体架构和目标
2. **配置开发环境** - 按照环境配置文档设置
3. **学习UI规范** - 掌握蓝色主题和设计标准
4. **按阶段开发** - 依次完成四个开发阶段
5. **运行测试** - 使用测试用例验证功能
6. **记录进度** - 更新进度追踪文档

### 账号切换
当需要切换开发账号时：
1. 查看进度追踪文档了解当前状态
2. 使用账号切换指南的提示词模板
3. 参考代码模板保持风格一致
4. 继续对应阶段的开发

### 代码质量保证
- 所有代码模板都经过语法检查
- 包含完整的导入语句和依赖
- 提供详细的注释说明
- 遵循Android开发最佳实践

## 📋 快速开始

1. 从 `01-环境配置.md` 开始配置开发环境
2. 阅读 `02-UI设计规范.md` 了解设计要求
3. 进入 `阶段开发/第一阶段-基础框架.md` 开始编码
4. 使用 `代码模板/` 中的模板确保代码质量
5. 参考 `测试用例/验收清单.md` 验证功能
