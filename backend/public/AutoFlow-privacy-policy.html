AutoFlow 隐私协议

说明 AutoFlow 如何收集、使用、储存和保护与您相关的数据。

一、收集的信息

为了保障 AutoFlow 自动化流程的稳定运行，会收集设备标识（设备 ID）、运行日志、渠道与激活信息、性能指标以及推送统计等数据。

收集行为仅限于分析稳定性、改善产品与合规要求，不会用于其他目的。

二、数据用途与存储

  • 设备 ID 等标识用于验证激活与会员有效期、控制调用频次。
  • 行为与性能日志帮助定位异常、优化脚本节点与流程。
  • 渠道信息（如友盟/极光）用于统计、推送与投放效果评估。
  • 全程采用必要的加密与权限控制，数据仅在合理范围内保存与使用，不会出售。

三、第三方 SDK

AutoFlow 需使用第三方 SDK 实现统计与推送，具体包括：

  • 友盟统计（UMeng）：官方隐私政策
  • 极光推送（JPush）：官方隐私政策

AutoFlow 仅在其允许范围内调用以上 SDK 的接口。

四、用户权利

  • 可通过系统设置随时撤销无障碍、通知等权限，撤销后部分功能可能受限。
  • 如需删除设备数据或取消绑定，请联系官方微信 xbdcc1，我们将在合理时间内协助处理。
  • 协议更新会在官网 autoflow.xbdcc.cn 公布，建议不定期查阅。

五、安全保障

采取多层安全策略（如数据加密、权限隔离、最小化存储、定期升级第三方依赖）确保运行环境合规。
